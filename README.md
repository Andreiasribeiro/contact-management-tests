# Contact Management Tests

This project was created during a Software Testing mentorship with Julio de Lima.

It demonstrates basic contact management functionality implemented in JavaScript and tested with automated unit tests using Mocha.

## Features

- Add a contact
- List contacts
- Find contact by name
- Remove contact
- Update phone number
- Count total contacts

## Technologies

- JavaScript (ES Modules)
- Node.js
- Mocha
- Mochawesome

contact-management-tests

## Running the project

Install dependencies:
npm install

Run the tests:
npm test

A test report will be generated using Mochawesome.
